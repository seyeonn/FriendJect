# FriendJect
### 주제
비대면 프로젝트도 대면하는 것처럼 개발 능률을 올려주고, 친근한
개발에 특화된 화상 채팅 프로그램 개발
# 기술 스택
### Front-end
- Vue.js
- openVidu
- BootstrapVue, Sass
### Back-end
- Java 8, Spring Boot, Spring Data JPA
- MySQL 5.7
- Postman
### DevOps
- AWS EC2, Docker, nginx, Jenkins
# 역할 배분
- 김주향 : 팀장, 풀스택, 미디어 담당, 프로젝트 총괄
- 이소라 : 풀스택, 발표담당
- 송진섭 : 풀스택, Notion 담당
- 안세연 : 풀스택, AWS 담당
- 정채은 : 풀스택, Jira 담당
